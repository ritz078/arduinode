Arduinode
---------

Data Acquisition System for a Hybrid/Electric Car.

Technologies Used
-----------------
* AngularJS
* NodeJS
* Johnny-five

Hardwares used
--------------
* Arduino UNO R3
* MPU 6050
* Sparkfun motor control shield

Features
--------

* Data shown on graphs read by MPU 6050
* 3d gyro visualisation on screen using three.js
* Motor controlling by keystrokes
